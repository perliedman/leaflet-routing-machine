Leaflet Routing Machine

This is a control for routing in Leaflet. In short, it helps finding the way from one location to another, possibly visiting a couple of via points in between. Behind the scenes, it uses OSRM to search the routes.

Drag and drop the blue waypoint markers, or grab the orange route line and drag a new via waypoint. You can also enter an address in the address text fields on the right side.

The plugin is free and open source. Downloads and documentation is available on Leaflet Routing Machine's GitHub page.

About this example

This is a basic example, where two waypoints have been added. Also, a geocoder has been added to support entering addresses, as well as finding the address for a location when a waypoint is drag-and-dropped on the map. It uses OpenStreetMap's Nominatim through Leaflet Control Geocoder, which works well with Leaflet Routing Machine.

The example also uses Leaflet.iconLabel to put the small letter indicators next to the waypoint markers in the map. 

L.Routing.control({
    waypoints: [
        L.latLng(48.8588,2.3469),
        L.latLng(52.3546,4.9039)
    ],
    geocoder: L.Control.Geocoder.nominatim(),
    plan: L.Routing.plan(null, {
        waypointIcon: function(i) {
            var label =
                String.fromCharCode(65 + i);
            return new L.Icon.Label.Default({
                labelText:
            });
        }
    })
}).addTo(map);
